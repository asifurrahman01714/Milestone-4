When You Will Use querySelector & querySelectorAll ?

  • When we will need to capture element in element
  • Example: let there are h1 in main article.So we can capture this by article.h1 in css. And this can be done by query Selector and querySelectorAll
  • If we use querySelector. This will capture only the first Element
  • But if we use querySelectorAll. This will give us a node list. Which is like an array
  • In that case we will capture the Elements by loop
